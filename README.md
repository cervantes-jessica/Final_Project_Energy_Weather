# Final_Project_Energy_Weather
STAT 386 Final Project

## Overview
This project analyzes how weather patterns influence wholesale electricity prices. Solely focusing on two price hubs: "Indiana Hub RT Peak" and "Palo Verde Peak"

## Data Sources
- U.S. Energy Information Administration (EIA)
- NOAA National Centers for Environmental Information (NCEI)
